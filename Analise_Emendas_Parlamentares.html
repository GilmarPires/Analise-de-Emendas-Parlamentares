Projeto de Anlise de Dados

O objetivo deste projeto analisar os custos de emendas parlamentares de 2024 que foram liberadas atravs do MINISTRIO DOS DIREITOS HUMANOS E DA CIDADANIA. Esse projeto foi todo elaborado em R, com a utilizao de pacotes como readr, dplyr, ggplot2, tidyr e readxl.

# Caminho para o arquivo baixado
file_path <- "Planilha_Emendas_ao_OGU_2024 - V1. Dados Abertos.xlsx"

# Ler a planilha do Excel, pulando a primeira linha
df <- read_excel(file_path, skip = 1)



# Filtrar as categorias indesejadas
df <- df %>%
  filter(!UF %in% c(NA, "No se Aplica", "PDT"))

# Converter a coluna Valor para numrica e ajustar para milhes de R$
df_milhoes <- df %>%
  mutate(Valor = as.numeric(Valor) / 1e6)

Analise de valores totais por UF

Aqui temos um grfico de barras com o valor total das emendas por UF, onde o destaque para o estado de So Paulo que recebeu o maior valor total em milhes de R$.

# Criar o grfico de barras com estilo personalizado e rtulos de valores
df_milhoes %>%
  group_by(UF) %>%
  summarise(total_emendas = sum(Valor, na.rm = TRUE)) %>%
  ggplot(aes(x = reorder(UF, total_emendas), y = total_emendas)) +
  geom_col(fill = "darkgreen") + # Cor das barras em verde escuro
  geom_text(aes(label = paste0("R$", round(total_emendas, 2))), hjust = 0, size = 2, color = "darkgreen") + # Rtulos em branco
  coord_flip() +
  labs(x = "", y = "Valor (Milhes de R$)", title = "Valor Total das Emendas por UF") +
  theme_minimal(base_size = 7) + # Aumentar o tamanho da fonte
  theme(
    panel.grid.major = element_blank(), # Remover linhas de grade maiores
    panel.grid.minor = element_blank(), # Remover linhas de grade menores
    panel.background = element_rect(fill = "lightyellow", color = NA), # Fundo amarelo fraco
    plot.background = element_rect(fill = "lightyellow", color = NA), # Fundo amarelo fraco
    text = element_text(color = "darkgreen", face = "bold"), # Rtulos em verde escuro negrito
    axis.text = element_text(color = "darkgreen", face = "bold"), # Rtulos dos eixos em verde escuro negrito
    axis.title = element_text(color = "darkgreen", face = "bold"), # Ttulo dos eixos em verde escuro negrito
    plot.title = element_text(color = "darkgreen", face = "bold", hjust = 0.5) # Ttulo do grfico em verde escuro negrito
  )

  group_by(Autor) %>%

Distribuio dos recursos por instituio

A grande maioria dos recursos so destinados a Instituies Privadas sem Fins Lucrativos.

# Agrupar os dados e somar os valores das emendas por Modalidade
df_instituicoes <- df_milhoes %>%
  group_by(`Modalidade`) %>%
  summarise(total_emendas = sum(Valor, na.rm = TRUE))

# Extrair os nmeros das variveis de Modalidade
df_instituicoes <- df_instituicoes %>%
  mutate(numero = sub(" - .*", "", Modalidade))

Concluso

Com base nesses dados podemos concluir que, esta pasta do Governo Federal destina a maior parte dos recursos das emendas parlamentares para instituies privadas sem fins lucrativos, com destaque para a rea de Sec. Nac. dos Direitos das Pessoas LGBTQIA+ que recebeu o maior valor total em milhes de R$ em 2024, o que acaba de beneficiar as instituies ligadas a essa pauta, recebendo mais recursos inclusive, do que a pauta dos direitos de crianas e dos adolescentes e a pauta dos direitos da pessoa idosa.
